# Automation

Based on https://github.com/gobanos/cargo-aoc

`cargo install cargo-aoc`

`cargo aoc credentials {token}`

# Usage

## get input data

`cargo aoc input -d 12 -y 2015`

## execute day

`cargo aoc -d 12`

## run benchmarks

`cargo aoc bench -d 12`

## run tests

`cargo test`
